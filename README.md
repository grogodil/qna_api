# qna_api

## Обзор проета
Этот проект реализует простой RESTful API для вопросов и ответов на языке Go. В качестве базы данных используется PostgreSQL, а для ORM — GORM. Управление миграциями реализовано через goose. Все сервисы контейнеризованы с помощью Docker и Docker Compose.

## Возможности
* Создание, получение и удаление вопросов
* Создание, получение и удаление ответов, связанных с вопросами
* Ответы можно оставлять только к существующим вопросам
* Один пользователь может оставить несколько ответов к одному вопросу
* Каскадное удаление: при удалении вопроса удаляются все его ответы
* Управление миграциями через goose
* Запуск и развертывание через Docker и Docker Compose

## Запуск проекта
1. Клонируете репозиторий
'git clone <ссылка-на-репозиторий'
'cd <название-папки>'
2. Соберите контейнер и запустите сервисы
'docker compose up --build'
3. API будет доступен по URL: http://localhost:8080

## API ендпоинты
* GET /questions/ — получить список всех вопросов
* POST /questions/ — создать новый вопрос
* GET /questions/{id} — получить вопрос со всем ответами на него
* DELETE /questions/{id} — удалить вопрос и все ответы на него
* POST /questions/{id}/answers/ — добавить ответ к вопросу
* GET /answers/{id} — получить конкретный ответ
* DELETE /answers/{id} — удалить ответ

## Тестирование
Для запуска тестов:
'go test -v ./internal/handler/...'
Тесты покрывают основные сценарии работы API и проверки валидации.
